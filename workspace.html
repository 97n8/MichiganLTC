Working rhythm

How to work in the workspace.

Start with the structure

Begin in /docs, /governance, and /templates before creating anything.

File by subject area

Place new working materials in the correct subject-area folder.

Template first

Use the templates before creating new trackers, notes, or decision records.

Register your sources

Record source support whenever claims, requests, or deliverables depend on external documents.

Keep governance current

Update the logs when decisions, assumptions, risks, escalations, or changes affect execution.

Placeholders over invention

When a source is missing, keep the placeholder — never invent content.

Need workspace access?

Access is granted to authorized participants only. Contact the network team to request or confirm authorization.
