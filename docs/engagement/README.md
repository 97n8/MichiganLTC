# Engagement

Use this folder for engagement plans, approved outreach materials, stakeholder coordination notes, and other relationship-management documents that support the Michigan LTC Network Resource Recovery Corridor.

## Recommended contents
- engagement plans
- outreach schedules
- approved talking points
- stakeholder lists with access controls as needed
- source-backed summaries of meetings or requests

## Placeholder discipline
If outreach language, letters, or relationship claims depend on missing source documents, note the gap clearly instead of inventing content.
