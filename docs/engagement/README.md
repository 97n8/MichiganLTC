## Repository boundary

Do not commit confidential agreements, private counterpart information, signatures, financial terms, or privileged legal comments to this repository. Store those records in the controlled project drive and reference only non-sensitive status metadata here.

## Placeholder discipline

If outreach language, letters, or relationship claims depend on missing source documents, note the gap clearly instead of inventing content.
